# MakaoLink

MakaoLink is a rental housing platform designed to simplify property discovery for tenants and streamline listing management for landlords and agents in Kenya.

The platform eliminates WhatsApp and Facebook listing chaos by providing a structured, verified, and subscription-based rental ecosystem.

---

## Vision

To build Kenya’s most trusted rental discovery and property listing platform.

---

## Tech Stack

### Frontend
- React (Vite)
- TailwindCSS
- React Router
- Axios
- Zustand (state management)
- React Hook Form
- Zod
- React Hot Toast

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Cloudinary (image storage)
- Resend (email notifications)
- Paystack / Daraja API (payments)

---

## User Roles

### 1. Tenant
- Browse listings
- Filter properties
- View property details
- Send inquiries

### 2. Landlord / Agent
- Create account
- Pay monthly subscription
- List properties
- Manage listings (CRUD)
- Receive inquiries via dashboard + email
- Reset password
- Contact admin

### 3. Admin
- Approve / reject listings
- Suspend accounts
- Verify payments
- Handle support queries
- Manage blog posts
- Verify landlord KYC
